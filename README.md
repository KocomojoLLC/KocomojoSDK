# KocomojoSDK

## Installation

KocomojoSDK is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```
pod 'KocomojoSDK'
```

## Cocoapod Kocomojo SDK Gem

`gem install cocoapods-kocomojo-sdk`

Then run: 

`pod kocomojo-sdk`.
